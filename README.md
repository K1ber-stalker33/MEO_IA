# MEO_IA
**Автор:** Рукомойкин Леонид
**Описание:** GUI-приложение «Expense Tracker» для отслеживания личных расходов с фильтрацией, подсчётом суммы за период
**Язык: Python

## Инструкция по запуску:
1. Установите Python 3.11.
2. Запустите приложение
   ```
python Expense Tracker.py
   ```
4. Используйте интерфейс длЯ для отслеживания личных расходов с фильтрацией, подсчётом суммы за период
5. Данные сохраняются в файл
